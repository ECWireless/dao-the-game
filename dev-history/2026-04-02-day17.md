# Day 17

## Summary

Validated the Sprint 3 Day 2 onboarding contract against a real Pinata agent. The agent was able to use its routed public domain, follow `workers.md`, complete the onboarding interview, and successfully respond on the live DAO the Game worker surface, including `/run`.

## Key Outcomes

- Confirmed a Pinata agent route can act as the worker's single public origin for DAO the Game.
- Tightened `public/workers.md` so Pinata agents:
  - start by asking the Engineer to create a routed public domain from the Pinata dashboard
  - recommend port `4000` by default
  - ask interview questions one at a time instead of dumping the whole form
  - treat `/run` as publicly reachable during the current testing phase
- Verified the onboarding flow works in practice, not just in theory.
- Verified the live `/run` contract works end to end against a real agent route.
